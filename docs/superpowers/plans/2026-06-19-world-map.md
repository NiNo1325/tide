# Carte du voyage (canvas, #3) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Une carte de voyage canvas animée (style SMB3) : 4 mondes, chemin de 11 nœuds, mascotte, tap = jouer.

**Architecture:** Tout dans `index.html`. Un drapeau `mapView` détourne `loop()` (dessine océan + `drawMap()`) et `pointerdown` (vers `mapTap`). Positions des nœuds calculées en zig-zag responsive. Bouton Retour en overlay DOM. Tout est dérivé de la logique de progression existante (`starsFor`/`levelUnlocked`/`nextLevel`).

**Tech Stack:** JavaScript vanilla, Canvas 2D. Pas de build, pas de test runner.

> **Vérification :** pas de framework de test. Après chaque tâche : `node --check` sur le script inline extrait, puis test **dans le navigateur**. Le canvas étant mis à l'échelle via `ctx.setTransform(DPR…)` et couvrant tout l'écran (`inset:0`), `e.clientX/clientY` = coordonnées de dessin directes.

```js
function mapTap(x,y){
  // flèches de monde (bande du haut)
  if(y<H*0.17){
    if(x<W*0.33){mapWorld=Math.max(0,mapWorld-1);return;}
    if(x>W*0.67){mapWorld=Math.min(DIFFS.length-1,mapWorld+1);return;}
    return;
  }
  for(let i=0;i<mapPos.length;i++){
    const p=mapPos[i];
    if(Math.hypot(x-p.x,y-p.y)<26){
      if(levelUnlocked(i,mapWorld)){
        mapView=false;$('mapBack').style.display='none';
        selTrack=i;selDiff=mapWorld;startGame();
      }else toast('Niveau verrouillé 🔒');
      return;
    }
  }
}
```
