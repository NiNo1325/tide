# Mode Duo (lanes) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ajouter un mode Duo (deux vagues, deux doigts) en généralisant le jeu en « lanes », le solo restant identique.

**Architecture:** On remplace les champs mono-vague de `game` (`nodes/genBeat/nextNode/trail`) par un tableau `game.lanes` (1 lane en solo, 2 en Duo). Les fonctions de jeu deviennent par-lane ; combo/score/perfects/total restent globaux. Duo = 5e difficulté `di=4` (`duo:true`) en Jeu libre, hors campagne.

**Tech Stack:** JavaScript vanilla, Canvas, Web Audio. Pas de build, pas de test runner.

> **Vérification :** `node --check` sur le script inline extrait, puis test navigateur. Le **mode solo doit rester identique** après Task 1 (régression nulle) — c'est le critère clé.
